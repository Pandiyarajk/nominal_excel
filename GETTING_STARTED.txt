🚀 QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

1. INSTALL DEPENDENCIES
   ─────────────────────
   pip install -r requirements.txt

   (On Linux/Ubuntu with externally managed Python:)
   pip install --break-system-packages -r requirements.txt


2. RUN THE APPLICATION
   ───────────────────
   streamlit run excel_app.py

   (Or use the provided script:)
   ./run_app.sh


3. OPEN YOUR BROWSER
   ─────────────────
   The app will open at: http://localhost:8501

🎯 BASIC WORKFLOW
═══════════════════════════════════════════════════════════════════════════════

1. Upload File
   └─ Click "Browse files" in sidebar
   └─ Select Excel or CSV file
   └─ Wait for success message

2. View & Edit
   └─ Go to "Edit Data" tab
   └─ Click cells to edit
   └─ Click "Save Changes"

3. Operations
   └─ Use "Operations" tab
   └─ Add/delete rows/columns
   └─ Sort by any column

4. Search & Filter
   └─ Use "Search & Filter" tab
   └─ Search text or filter by values
   └─ Save filtered results

5. Analyze
   └─ Go to "Analytics" tab
   └─ View statistics and charts
   └─ Explore correlations

6. Export
   └─ Click "Download Excel" in sidebar
   └─ Save your changes


💡 PRO TIPS
═══════════════════════════════════════════════════════════════════════════════

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

  App won't start?
  └─ Check dependencies are installed
  └─ Verify port 8501 is available
  └─ Try: streamlit run excel_app.py --server.port 8502

  File won't upload?
  └─ Check file format (.xlsx, .xls, .csv)
  └─ Verify file isn't corrupted
  └─ Try with sample files first

  Changes not saving?
  └─ Click "Save Changes" button
  └─ Check browser console (F12)
  └─ Refresh and try again
